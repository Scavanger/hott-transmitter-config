Prerequisites
=============
- install Java 8 SDK (1.8.0_111 or later)
- install Apache Maven (3.1.1 or later)
- install git (1.8.3 or later)

Clone repositories
==================
$ git clone https://gitlab.com/mdlviewer/hott-transmitter-config.git

Build and install application
=============================
$ cd hott-transmitter-config
$ mvn install
